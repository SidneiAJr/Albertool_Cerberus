# 🐺 Albertool Cerberus

Monitor de servidor e rotas diretamente na sidebar do VS Code — sem precisar abrir o terminal.

## ✨ Funcionalidades

- 🟢 Monitor de servidor online/offline em tempo real
- 📡 Lista todas as rotas do projeto automaticamente
- 🎨 Cores por método HTTP (GET, POST, PUT, DELETE)
- 📋 Clique na rota para copiar a URL completa
- 🌐 Rotas GET abrem direto no browser
- 🔄 Atualização automática a cada 5 segundos
- ⚡ Refresh ao salvar qualquer arquivo

## 📦 Como usar

1. Abra um projeto com rotas Express
2. Clique no ícone 🐺 na barra lateral
3. Inicie seu servidor normalmente
4. Cerberus detecta automaticamente as rotas e o status

## ⚙️ Comandos

| Comando | Descrição |
|---------|-----------|
| `Cerberus: Atualizar` | Atualiza rotas e status |
| `Cerberus: Mudar Porta` | Muda a porta monitorada |
| `Cerberus: Copiar Rota` | Copia URL completa |

## 🔧 Frameworks suportados

- Express (JavaScript / TypeScript)

## 🔧 Requisitos

- VS Code 1.85.0+
